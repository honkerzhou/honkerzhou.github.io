# 前端学算法

视频、书籍、练习题三者相结合，视频推荐B站上的JavaScript数据结构与算法 (opens new window)，书籍推荐《学习JavaScript数据结构与算法》（第3版），练习题推荐LeetCode精选TOP面试题 (opens new window)
分类进行顺序学习，比如先看视频中栈相关的课程，立马去看书中相关内容，再练习LeetCode上相应标签的题目。
刚开始自己直接看书，总有地方看不懂（说明书写的还不够好），也坚持不下去，因为没有做题目，没有正向激励。后来看完书中一类内容（比如树）后去做LeetCode上的题目，比起之前确实更容易坚持一点了。但是总有一些知识点搞不懂，书中写的不明白，到了红黑树的时候真的看不下去了，找到了这个视频，才发现视频-书-题，这样的顺序，自己开心多了。

由于视频中没有红黑树的代码，所以参考了书中代码，比较后发现，红黑树变换规则中只讲了LL和LR的情况，没有讲RR和RL的情况。搞得一直没有理解书中代码的逻辑（无比相信视频） [2020]有个误区提醒大家：红黑树_变换规则四、红黑树_变换规则五中只讲了LL和LR的情况，没有讲RR和RL的情况。所以规则四和五都可以再分别延展出一种情况（RL和RR），总计七种情况。看完视频后可以结合《学习JavaScript数据结构与算法》（第3版）中相关代码理解~
