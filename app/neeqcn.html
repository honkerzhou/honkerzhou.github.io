# 长按复制文本

今天运营大哥说APP里有几个页面要可以长按复制文本，方便用户转发文字，才发现自己写的APP里真的没这功能！！之前做的复制文本都是点击按钮复制相应文本的。。。

# 怎么做

给包裹文本的标签加一句css代码：-webkit-user-select: text;

# 例子

<!-- 假设你的HTML是这么写的 -->
<div class="text">长按复制我</div>
1
2
/*相应的CSS*/
.text {
    -webkit-user-select: text;
}
1
2
3
4

# 为什么

mui默认禁止选择复制文本，mui.css中默认有以下代码：

*{
    -webkit-box-sizing: border-box;
            box-sizing: border-box;

    -webkit-user-select: none;

    outline: none;

    -webkit-tap-highlight-color: transparent;
    -webkit-tap-highlight-color: transparent;
}
