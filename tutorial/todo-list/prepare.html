# 几句交代

TIP

由于每个人的实际情况不一样，并且安装之法门Google一下多如牛毛，故在这里不详细讲解安装过程，只附上各自的官网安装步骤方便大家正确安装与学习，如在实际操作过程中遇到问题，欢迎大家留言互动！

建议大家尽量选择最新的长期支持的稳定版本，那么运行本教程肯定是没有问题的。有特殊版本要求的我会特别说明，当然，如果能和本教程所使用的的版本一致自然最好不过(括号中为本教程所使用版本)。

# 动手操作

  1. 安装Node.js(12.16.1) (opens new window)

  2. 安装Git(2.26.0) (opens new window)

  3. 安装MongoDB(4.2.0) (opens new window)
    这里多说两句，也是自己当时遇到的坑：
    这都2019年了，随手百度一下MongoDB安装教程，全是四五年前的，什么安装完后还要配置mongod.cfg文件、MongoDB系统服务之类的，现在MongoDB都4.0了，安装程序都自带了好么！所以，跟着官方安装教程 (opens new window)，才是最妥的！另外Windows下安装的时候不要勾选Compass，不然会耗时到怀疑人生，以后有需要再另行下载。

  4. 安装Vue CLI：

    npm install -g @vue/cli
# 安装结束后，使用该命令验证是否安装成功,本教程的版本是4.2.3

 # 进入server文件夹，初始化后端代码
 cd todo-list-server
 npm init -y

 # 安装koa
